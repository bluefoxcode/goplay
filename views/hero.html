This is a very simple to use REST style API built in go with postgresql db. It displays a good use of mux, unrolled/render, JSON manipulation, good separation of responsibilites and code organization.

List Heroes

curl http://goplay-api.bluefoxcode.com/hero

Show Hero

curl http://goplay-api.bluefoxcode.com/hero/2

Post Hero

curl -X POST -H "Content-Type: application/json" -H "Cache-Control: no-cache" -d '{ "description":"This hero is so awesome! You can tell by his cape.","name":"Capey McCaperson"}' "http://goplay-api.bluefoxcode.com/hero"
